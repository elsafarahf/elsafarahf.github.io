h.cal
Code Format :  $ ncal -h

ii. the improvement that your commands introduce 
We extract only working days from the calendar while also removing all unnecessary data such as empty lines and month headings.
Code Format :  $ ncal -h | grep -vE "^S|^ |^$"

We can also remove all alphabetic characters.
Code Format :  $ ncal -h | grep -vE "^S|^ |^$" | sed "s/[[:alpha:]]//g"

The above integers are all working days for a current month. We format and sort them.
Code Format :  $ ncal -h | grep -vE "^S|^ |^$" | sed "s/[[:alpha:]]//g" | fmt -w 1 | sort -n

If you wish to calculate the number of working days for a current month simply pipe the output to “wc” command.
Code Format :  $ ncal -h | grep -vE "^S|^ |^$" | sed "s/[[:alpha:]]//g" | fmt -w 1 | sort -n | wc -l

We just want to see the dates for some day of the week.
Code Format :  $ ncal | grep Th
